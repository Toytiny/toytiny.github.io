AutoTrack: Towards High-Performance Visual Tracking for UAV with Automatic Spatio-Temporal Regularization

Download this paper    Download the code
Publishd by CVPR2020
Yiming Li, Changhong Fu, Fangqiang Ding, Ziyuan Huang, and Geng Lu
In this paper, local response variation indicates local credibility, thus restricting local correlation filter learning. Global variation is able to control how much the correlation filter learns from the whole object. In addition, we try to bridge the gap between the theory and practice by utilizing visual tracking in UAV localization in the real world.

DR^2Track: Towards Real-Time Visual Tracking for UAV via Distractor Repressed Dynamic Regression

Download this paper    Download the code
Published by IROS2020
Changhong Fu, Fangqiang Ding, Yiming Li, Jin Jin, and Chen Feng
Different from the classical DCF trackers with a fixed training label, the regression target response in this work can be dynamically changed to upgrade the training quality. By introducing local maximums of response maps, distractors can be located and repressed with pixel-level penalization.

Automatic Failure Recovery and Re-Initialization for Online UAV Tracking with Joint Scale and Aspect Ratio Optimization

Download this paper    Download the code
Published by IROS2020
Fangqiang Ding, Changhong Fu, Yiming Li, Jin Jin, and Chen Feng
In this work, a novel UAV tracking framework of joint scale and ARC estimation is proposed. Also, an object proposal based re-detection algorithm is introduced to achieve long-term tracking. Experimental comparison with 30 SOTA trackers exhibits the superiority of our method.

Augmented Memory for Correlation Filters in Real-Time UAV Tracking

Download this paper    Download the code
Published by IROS2020
Yiming Li, Changhong Fu, Fangqiang Ding, Ziyuan Huang, and Jia Pan
In this work, augmented memory for correlation filters is proposed. Essentially, augmented memory maintains a FIFO queue to store distinct previous views. Each stored view in the memory will be assigned the desired response. Compressed context learning provides more negative samples and suppresses responses to surrounding areas of the tracked object.

ADTrack: Target-Aware Dual Filter Learning for Real-Time Anti-Dark UAV Tracking

Download the code
Accepted by ICRA2021
Bowen Li, Changhong Fu, Fangqiang Ding, Junjie Ye, and Fuling Lin
This work puts forward a novel real-time tracker with anti-dark function, i.e., ADTrack. ADTrack first implements image enhancement and target-aware mask generation based on an image enhancer. With the mask, ADTrack innovatively proposes dual filters, i.e., the target-focused filter and the context filter, regression model. In addition, the first dark tracking benchmark, UAVDark70, is also constructed in this work for visual tracking community.

Mutation Sensitive Correlation Filter for Real-Time UAV Tracking with Adaptive Hybrid Label

Download the code
Accepted by ICRA2021
Guangze Zheng, Changhong Fu, Junjie Ye, Fuling Lin, Fangqiang Ding
In this work, a mutation sensitive and resistant tracker with an adaptive hybrid label is proposed for real-time UAV tracking. As a novel evaluation of response map introduced in the AHL, MTF indicates the severity of tracking mutations. The whole work-flow also adopts the genetic idea to reach optimal solutions. Comprehensive experiments have corroborated the favorable performance of MSCF. Furthermore, employing MSCF for UAV tracking in the real-world environment is also taken into consideration to conduct in the future.

Learning Dynamic Regression with Automatic Distractor Repression for Real-Time UAV Tracking

Download this paper    Download the code
Published by EAAI
Changhong Fu, Fangqiang Ding, Yiming Li, Jin Jin, and Chen Feng
This work proposes to monitor the local maximums in the response map. In consideration of the fast motion of UAV, we present a simple but useful search area decision strategy to tackle the abrupt tremble of UAV or acceleration of the object. Specifically, we design a shifting-cropping-filling scale sample reuse method to elevate the tracking speed without losing accuracy.

Spatial Reliability Enhanced Correlation Filter: An Efficient Approach for Real-Time UAV Tracking

Download the code
Accepted by TMM
Changhong Fu, Jin Jin, Fangqiang Ding, Yiming Li, and Geng Lu
In this work, a novel spatial reliability enhanced learning strategy is proposed to boost the tracking performance. Appearance variations within the searching area can be effectively detected by the spatial reliability map. A novel response pool and updating strategy are proposed to provide accurate reference for reliability analysis.

Multi-Regularized Correlation Filter for UAV Tracking and Self-Localization

Download the code
Submitted to TIE
Junjie Ye, Changhong Fu, Fuling Lin, Fangqiang Ding, Shan An, Geng Lu
This work proposes a multi-regularized correlation filter, i.e., MRCF tracker. It can smooth the deviation of responses adaptively and optimize channel weight distributions jointly. Comprehensive experiments on multiple UAV tracking benchmarks show that MRCF performs prominently against other SOTA trackers, meeting the real-time requirement of UAV applications. Moreover, an original solution to UAV selflocalization is provided based on our MRCF.

All-Day Object Tracking for Unmanned Aerial Vehicle

Download this paper    Download the code
Submitted to TIP
Bowen Li, Changhong Fu, Fangqiang Ding, Junjie Ye, Fuling Lin
This work puts forward a novel real-time tracker with illumination adaptation and anti-dark function. Besides, an innovative dual filter regression model is proposed, in which the dual filters restrict each other in training and compensate each other in detection. In addition, the first large-scale dark tracking benchmark, UAVDark135, is also built in this work for visual tracking community. We strongly believe that the proposed tracker and dark tracking benchmark will make outstanding contribution to the research of UAV tracking in all-day conditions.

Correlation Filter for UAV-Based Aerial Tracking: A Review and Experimental Evaluation

Download this paper   Download the code
Submitted to GRSM
Changhong Fu, Bowen Li, Fangqiang Ding, Fuling Lin, and Geng Lu
This work firstly introduced the tracking scenes onboard UAV, the uniqueness and challenge of UAV tracking. Then, the DCFbased tracking algorithms’ common basic architecture is proposed for overall understanding. Thirdly, this work introduces the highlight of each DCF-based tracker. Lastly, exhaustive experiments were conducted on six UAV tracking benchmarks to evaluated all mentioned DCF-based trackers and demonstrate the superiority of their tracking.
